# Sunrise Hospital EMR System

A secure cloud-ready Electronic Medical Record (EMR) system 
built with Python and PostgreSQL.

## Features
- Secure login with password hashing (bcrypt)
- Real patient data management with PostgreSQL
- Age-based patient filtering
- Real-time clinical analytics and visualizations
- Complete audit logging of all user actions

## Technologies Used
- Python
- PostgreSQL
- psycopg2
- bcrypt
- python-dotenv
- matplotlib
